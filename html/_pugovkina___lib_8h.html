Заметки
Сначала рисуются лепестки, начиная с верхнего левого, затем рисуется серединка.

Рисует солнце с лучиками

Аргументы
xx - Х-координата центра солнца
yy - Y-координата центра солнца
sizeXsizeX - изменение размера по координате Х
sizeYsizeY - изменение размера по координате Y
eyeseyes - изменение размера глаз
lenghLuchlenghLuch - длина луча
smilesmile - ширина улыбки
sunColorsunColor - цвет солнца
Заметки
Солнце рисуется с помощью эллипса. Радиус солнца - 30 пикселей. У солнца - 8 лучиков.
Для рисования носа используется полигон.
Для рисования рта и глаз используется эллипс.
При изменении значения аргумента smile (-3 до 1) можно сделать, чтобы солнце - "разговаривало"
При изменении значения аргумента eyes (-2.5 до 2.5) можно сделать, чтобы солнце - "мигало"

Выводит надпись на экран.
Надпись выводится снизу вверх, состоит из 2-х строчек.
Текст типа Arial, размер шрифта - 100

Аргументы
fon_Colorfon_Color - задаёт цвет фона
text_Colortext_Color - задаёт цвет текста
text_first_string[]text_first_string[] - выводит текст на первой строчке
text_second_string[]text_second_string[] - выводит текст на второй строчке
Заметки
Текст выводится по центру относительно координаты Х = 550.
Текст появляется снизу экрана, начиная с координаты по Y
для первой строки: Y = 700,
для второй строки: Y = 800
и движется вверх по координате Y, до Y = 350 (первая строчка)
См. также
txSetTextAlign ()
Пример использования:
Titles (RGB (89, 184, 33), TX_WHITE, "Мультфильм", "Прогулка c подружками");
